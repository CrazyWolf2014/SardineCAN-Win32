Sardine CAN - Windows DLL version 0.2 alpha (Copyright Olaf @ Hacking volvo blog (hackingvolvo.blogspot.com). E-mail: hackingvolvo@gmail.com

This is Sardine CAN, an open source J2534 device, which makes it possible to connect various diagnostic programs (eg. VIDA from Volvo) to your car.

Files in this repository are only for the Windows DLL. You also need Arduino firmware ( available in github.com/hackingvolvo/SardineCAN-Arduino )

Installation instructions can be found in INSTALLATION.txt
